This is a simple Todo List where you can see all your tasks, add some and delete some.

## Installation

- Clone this repo :
  
      git clone {url}

- Install packages :

       cd ./front && npm i  
       cd ./back && npm i  

- Create .env and modify it using .env.sample

- Create database : 

        cd ./back
        npm run db:migrate

- Run App :

        cd ./front
        npm run dev
      
        cd ./back
        npm run dev  

## Packages :
